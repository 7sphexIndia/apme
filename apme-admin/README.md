# apme-admin (APME dashboard)

Separate Vite app in this repo for gallery management.

## Run

1. Start PHP API: `php -S 127.0.0.1:8080 -t apme-backend` (or your host).
2. From repo root: `npm run dev:admin` **or** `cd apme-admin && npm run dev`
3. Open **http://localhost:5174** → `/login`

Main public site: **http://localhost:5173**.
